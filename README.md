## âš™ï¸ Getting Started

### 1. Clone the Repository

```bash
git clone git@github.com:abdu1478/NovaProperties.git
cd NovaProperties
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Create a `.env` file

Inside the root of the project:
